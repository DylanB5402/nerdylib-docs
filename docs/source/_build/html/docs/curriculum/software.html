Recommended/Required Software¶

Required Software¶

Software you have to install on your personal devices

  • VS Code + WPILib Software

  • GitKraken (or alternative Git client, if desired)

Recommended Software¶

Additional software that’ll make your life easier

  • FRC Update Suite

  • Logger Pro (ask a programming lead for a copy)

  • CTRE Phoenix Toolsuite

  • Python

  • Jevois software (if you’re doing vision)

  • Limelight software (if you’re working with that)

Required Accounts¶

  • Github

  • Github Student Pack

  • Slack
